Features
--------
- Password Protection
- Multi-user Support
- Attractive UI
- Students Management
- Teachers Management
- Student Active/Inactive Management
- Attendance Management
- Exam Result Management
- Students & Teachers Payment Management
- Payment Receipt Print
- Payment Viewer
- Reports
    - Student Report
    - Payment Report
    - Cost Report
    - Finance Report
- Logs (For Admin) 
	- New Student Add
	- Student Info Changed
	- New Teacher Add
	- Teacher Info Changed
	- Password Changed
	- Add Attendance
	- Add Exam Result
	- Upgrade One Class
	- Make Active/Inactive
	- Some Window Open Log
- Easy to use
- Database Backup/Restore
